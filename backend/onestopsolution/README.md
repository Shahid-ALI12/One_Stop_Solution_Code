# onestopsolution

> Scaffolded with uv_init.py | Type: fastapi

## Setup
```bash
uv sync
```

## Run
```bash
uv run main.py
```

## Add a package
```bash
uv add package-name
```

## Refresh requirements.txt
```bash
uv export --format requirements-txt --no-hashes > requirements.txt
```

## API Docs
After running, visit http://localhost:8000/docs
